====================================================================
🔑 НОВАЯ АВТОРИЗАЦИЯ - КАК ПОДКЛЮЧИТЬСЯ К POCKET OPTION
====================================================================

ШАГ 1: ОТКРОЙ POCKET OPTION
----------------------------
1. Открой обычный браузер (Chrome, Firefox, Edge)
2. Перейди на: https://m.pocketoption.com/ru/
3. Войди в свой аккаунт (демо или реальный)

ШАГ 2: ИЗВЛЕКИ ТОКЕН (SSID)
----------------------------
СПОСОБ A - Через extract-token.html (РЕКОМЕНДУЕТСЯ):
1. Открой файл: extract-token.html (в том же браузере где открыт Pocket Option)
2. Нажми F12 → Console
3. Скопируй ВСЕ cookies: введи document.cookie и нажми Enter
4. Скопируй результат и вставь в extract-token.html
5. Нажми "Извлечь SSID токен"
6. Нажми "Копировать токен"

СПОСОБ B - Вручную:
1. Нажми F12 в браузере где открыт Pocket Option
2. Перейди во вкладку Console
3. Введи: document.cookie
4. Найди часть где написано: ssid=XXXXXXXXX
5. Скопируй только значение после ssid= (до следующей ;)

ШАГ 3: ПОДКЛЮЧИ В АНАЛИЗАТОРЕ
----------------------------
1. Открой анализатор (index.html)
2. Нажми кнопку "🔑 Войти по токену"
3. Вставь скопированный SSID токен
4. Нажми "✅ Подключить"

ГОТОВО! ✅
----------------------------
Теперь анализатор подключен к Pocket Option через прокси сервер!

====================================================================
💡 СОВЕТЫ:
====================================================================

• Токен работает пока ты не выйдешь из Pocket Option
• Если отключилось - просто введи токен заново
• Прокси сервер должен быть запущен (порт 3000)
• Для автоторговли нужно подключиться по токену

====================================================================
⚠️ ВАЖНО:
====================================================================

• Никому не показывай свой SSID токен!
• Токен дает доступ к твоему аккаунту
• После выхода из Pocket Option токен перестает работать
• Для демо и реального счета нужны разные токены
